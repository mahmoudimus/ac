First, go to the directory from which you started the VM, and run the following to connect to the VM: 
vagrant ssh
### Stable versions To start a local JIRA/Confluence server with Connect matching what is currently in production in Atlassian Cloud, run the following from within the VM: #### JIRA 
atlas-run-connect --product jira
#### Confluence 
atlas-run-connect --product confluence
### Bleeding edge

Experts only!

This is using the latest development version of Connect, so you can test the latest and greatest, and as such may be unstable.

To start a local JIRA/Confluence server with the latest development version of Connect (bleeding edge), run: #### JIRA 
atlas-run-connect --product jira --beta
#### Confluence 
atlas-run-connect --product confluence --beta
