The Vagrantfile in this directory can be used to create a VM from the packaged box uploaded on VagrantCloud.
The Vagrantfile in the provision/ directory is used to create a VM from hashicorp/precise32, then run the provisioning
scripts which install Java, the Atlassian Plugin SDK, Node.js, etc.

For more details: https://extranet.atlassian.com/display/ARA/Vagrant+box
